Room
-데이터베이스를 쉽게 사용하기 위한 API
안드로이드 데이터 저장방식
-Key-value : SharedPreference (내장)
-관계형 방식 : SQLlite (내장)
-Sqlite를 직접이용하는게아닌 Room을 사용해서 Sqlite를 이용한다
-Room을 얻어와서 Room한테 명령을하면 Room이 Sqlite를 조작한다(추상화계층)
안드로이드의 내장된 관계형 데이터베이스를 사용시 장점(서버로부터 데이터를 가져오는게아닌 안드로이드에서 직접저장)
1.서버로부터 데이터를 주고 받는다
요청 : 앱 > 서버 > 데이터베이스(관계형)
응답 : 데이터베이스 > 서버 > 앱(관계형)
서버로부터 온 데이터베이스를 앱데이터베이스에 저장가능!
캐시(Cache)
캐싱(Cashing) : 캐시를 남기는 작업 (캐싱을 사용하면 한번 서버로부터받은 정보들은 앱에 저장해서 다음에 네트워크 연결이없어도 볼 수있음)
